📖 ❓ HKDSE Reading - Question Types

The HKDSE reading test has 5 different types of questions. Each type has a different way to answer. Learn all 5 and you will improve your score.

Overview: 5 Question Types

Type What Is It? Difficulty How Many?
Multiple Choice Choose A, B, C, or D Easy-Medium 10-15
Short Answer Write 1-3 words Easy-Medium 5-10
Gap Filling Fill in missing words in sentences Medium 5-10
Matching Match information from different parts Medium-Hard 3-8
Summary / Completion Complete a summary in your own words Hard 2-5

Question Type 1: Multiple Choice

🎯 What Is It?

You read a question and choose one answer from 4 choices (A, B, C, D). Only ONE answer is correct.

How to Answer Multiple Choice Questions:

  1. Read the question completely. Understand what it's asking.
  2. Find keywords in the question. These are important.
  3. Scan the passage for these keywords. Find the relevant sentence.
  4. Read the sentence carefully. Get the information you need.
  5. Check each answer choice:
    • A) Is this correct? Mark if yes.
    • B) Is this correct? Mark if yes.
    • C) Is this correct? Mark if yes.
    • D) Is this correct? Mark if yes.
  6. Choose the BEST answer. Usually only 1 is perfect.
  7. Move on. Don't waste time second-guessing.

✓ Example Multiple Choice Question:

Passage text: "The library is open from 9 AM to 6 PM on weekdays. On weekends, it opens at 10 AM and closes at 4 PM."

Question: When does the library open on Saturday?

A) 9 AM
B) 10 AM ← CORRECT
C) 4 PM
D) 6 PM

Why? The passage says "On weekends, it opens at 10 AM." Saturday is a weekend day.

Question Type 2: Short Answer

🎯 What Is It?

You answer with a short phrase - usually 1-3 words. You write your own answer, not choose from options.

How to Answer Short Answer Questions:

  1. Read the question. Know exactly what information to find.
  2. Scan for keywords. Find them in the passage.
  3. Find the answer - it's usually right there! Look in the sentence with the keyword.
  4. Copy the answer from the passage. Use the exact words if possible.
  5. Keep it SHORT (1-3 words). Don't write extra words.
  6. Check grammar and spelling quickly. Spelling mistakes might lose points.

✓ Example Short Answer Question:

Passage text: "John studies mathematics at Oxford University. He has been there for 2 years. He enjoys working with numbers and solving difficult problems."

Question: What does John study?

Answer: Mathematics

Why? The passage says "John studies mathematics." This is exactly what the question asks. 1 word = short answer ✓

❌ Common Short Answer Mistakes:

  • Writing TOO MUCH. The question asks for 1-3 words, write 5 words = WRONG
  • Changing words from the passage. Usually the exact phrase is correct.
  • Misspelling. "Mathematics" not "mathmatics"
  • Writing a full sentence. Keep it SHORT!

Question Type 3: Gap Filling

🎯 What Is It?

Sentences have blank spaces (___). You fill in the missing word or phrase from the passage. Sometimes they give you a word bank (list of words to choose from).

How to Answer Gap Filling Questions:

  1. Read the sentence with the blank. What word is missing?
  2. Look at the grammar around the blank. Should it be a noun? Verb? Adjective?
  3. Scan the passage for similar information. The missing word is usually nearby.
  4. If there's a word bank, check each word:
    • Does it fit grammatically? (right part of speech)
    • Does it make sense in the context?
    • Does it appear in the passage?
  5. Fill in the blank.
  6. Read the whole sentence - does it make sense now?

✓ Example Gap Filling:

Passage: "Last summer, Maria traveled to Japan. She visited Tokyo, Kyoto, and Osaka. She enjoyed Japanese food very much."

Question: Gap Filling - Complete the sentence:

"Maria visited three _______ in Japan."

Word bank: countries / cities / temples / restaurants

Answer: cities

Why? Passage says "Tokyo, Kyoto, Osaka" - these are cities. They are not countries, temples, or restaurants. Grammatically: "three cities" is correct. ✓

Question Type 4: Matching

🎯 What Is It?

You match items from one list (Column A) with items in another list (Column B). One side describes things, the other side has options to match with.

How to Answer Matching Questions:

  1. Read all items in Column A. Understand what each one is.
  2. Read all items in Column B. Know your options.
  3. For each item in Column A, find the matching description in Column B:
    • Is it a name → match with a description
    • Is a reason → match with a result
    • Is a place → match with its characteristics
  4. Scan the passage for confirmation. Make sure your match is correct.
  5. Use elimination. Once you match one item, cross it off. Don't use it again.
  6. Difficult items last. Answer easy matches first, then hard ones.

✓ Example Matching:

Column A (People):
1. John
2. Sarah
3. Mike

Column B (Jobs):
A. Teacher
B. Doctor
C. Engineer

Passage: "John works as a teacher in a school. Sarah is a doctor at the hospital. Mike is an engineer and designs buildings."

Answer:
1. John → A (Teacher)
2. Sarah → B (Doctor)
3. Mike → C (Engineer)

Question Type 5: Summary / Completion

🎯 What Is It?

A summary of the passage is given with some blanks. You must write a word or short phrase in your own words (not copy from the passage) to complete the summary. This is the HARDEST question type.

How to Answer Summary Questions:

  1. Read the whole passage first. Understand the main idea.
  2. Read the summary. Understand what information is missing.
  3. For each blank, understand:
    • What type of information goes here? (fact, date, reason, etc.)
    • What's the main idea of this part?
  4. Find the relevant section in the passage. Scan for related information.
  5. Write your own words. Don't copy exactly (unless instructed).
  6. Check:
    • Does it fit grammatically?
    • Is it accurate?
    • Is it NOT copying the exact phrase?

✓ Example Summary:

Passage: "Electric cars are becoming more popular around the world. They produce no pollution and are cheaper to operate than gasoline cars. However, the main problem is that batteries are expensive and take a long time to charge."

Summary to complete:
"Electric cars have many benefits, such as producing (1) _______ and being cost-effective. However, they have disadvantages, including (2) _______ batteries and slow charging times."

Answers:
(1) no/zero pollution (or: clean energy, no emissions)
(2) expensive/costly (or: high-cost, expensive)

Why? We found the information in the passage and wrote it in our own words. Not exact copies, but accurate. ✓

Question Type Comparison

Type Answer Style Time Per Q Difficulty Key Skill
Multiple Choice Choose from 4 options 30-45 sec Easy-Medium Scanning
Short Answer Write 1-3 words 1-2 min Easy-Medium Finding facts
Gap Filling Write 1 word/phrase 1-2 min Medium Grammar + Context
Matching Connect two lists 2-3 min Medium-Hard Understanding relationships
Summary Write your own words 3-5 min Hard Understanding main ideas

Strategy: Question Types in Order of Difficulty

Do questions in this order (easiest first):

  1. Multiple Choice (1st) - These are fastest. Get these done quickly!
  2. Short Answer (2nd) - Also quick. Just find facts in the passage.
  3. Gap Filling (3rd) - Requires more thinking about grammar.
  4. Matching (4th) - Takes more time because you need to understand relationships.
  5. Summary (5th, LAST) - Hardest. Save this for when you have time to think carefully.

This order helps you get points from easier questions first!

Common Mistakes for Each Type

❌ Multiple Choice:

  • Reading the choices but not the passage
  • Choosing an answer without checking the passage
  • Overthinking - sometimes the first answer is correct

❌ Short Answer:

  • Writing too many words (should be 1-3)
  • Poor spelling or grammar
  • Changing words when you should copy from passage

❌ Gap Filling:

  • Not considering grammar (singular/plural, verb tense)
  • Using words that don't appear in the passage
  • Writing more than one word when only one is needed

❌ Matching:

  • Using one item twice
  • Guessing without reading passage
  • Not understanding what each column represents

❌ Summary:

  • Copying word-for-word from passage (they want your own words!)
  • Paraphrasing wrongly and changing the meaning
  • Writing answers that are too long

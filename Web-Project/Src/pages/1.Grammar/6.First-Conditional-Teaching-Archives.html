📦 Concept Compressed

The First Conditional expresses real and possible future situations and their likely results. It uses the Present Simple in the if clause and will (or other modals) + base verb in the main clause.

Structure

If + subject + present simple, subject + will/modal + base verb

When to Use

  • Real future possibilities: If I have time, I will visit you.
  • Predictions: If you do not hurry, you will miss the bus.
  • Promises/threats: If you help me, I will buy you dinner.

Modal Variations

Modal Example
can If you finish early, you can leave.
might If it rains, we might stay home.
should If you feel sick, you should see a doctor.

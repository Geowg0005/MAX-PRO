📦 Concept Compressed

The Present Continuous (also called Present Progressive) describes actions happening at the moment of speaking or ongoing temporary situations. It uses am/is/are + verb-ing.

Formation

  • Structure (Affirmative): Subject + am/is/are + Verb-ing + Object
  • Structure (Negative): Subject + am/is/are not + Verb-ing + Object
  • Structure (Question): Am/Is/Are + Subject + Verb-ing + Object?

Spelling Rules for -ing

  • Most verbs: Add -ing (play → playing)
  • Verbs ending in -e: Drop -e, add -ing (make → making)
  • Short verbs (CVC pattern): Double final consonant (run → running, sit → sitting)
  • Verbs ending in -ie: Change to -y (lie → lying, die → dying)

Common Time Markers

Now, right now, at the moment, currently, today, this week, these days
